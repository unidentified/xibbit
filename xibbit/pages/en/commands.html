Basic commands

Please note: Refer to the FAQ for more information on XMPP and usage of this page.

/join [room]
Example 1: /join room1 (joins room1 on the currently used chatserver).
Example 2: /join room1@conference.someserver.com (joins room1 on server conference.someserver.com - please note that not all XMPP servers support joining rooms on other servers, so joining a room at another server than you're registered and and currently using may fail).

/part
Leave the current room. (Please note: At the moment, you have to close private chats by closing the tab - click on the "x").

/names
List all users in the current room. Use this on mobile to get as user list, as there is no userlist displayed for very small screens.

/query [nick]
Example: /query user1 (starts a private chat with the user using the nickname user1).

/ignore [nick]
Example: /ignore user1 (ignore user1 in this room).

/unignore [nick]
Example: /unignore user1 (unignore user1 in this room, so you read their messages again).

/themes
List all available chat themes.

/theme [theme]
Example: /theme light
Load a theme. List themes with /themes.

/me [message]
Is shown as "{your username} message" and your message is in italics.
Example: /me is away for a while
If your username is user1, this is shown as "user1 is away for a while".

/help
Prints out an URL to the command file.

Administrative commands

Please note: Refer to the informational page for more information on XMPP and administration of a groupchat room.

/config
Configure the current room.

/set {affiliation | role} [value] [user]
Set a users affiliation or role to a certain value. "user" can be the nickname if the user is present in the current room, or their JID.
Example 1: /set affiliation member test1@myserver.com (sets the users with the JID test1@myserver.com as a member in the current room).
Example 2: /set affiliation admin user1 (sets the occupant with the nickname user1 in the current room as admin).
Example 3: /set role moderator user2 (sets the role of the occupant with the nickname user2 in the current room to moderator, i.e. user2 is moderator as long as in room now).

/list [owner | admin | member | outcast | moderator | participant]
List the users with a certain affiliation or role.
Example 1: /list admin (lists all users with the affiliation admin in the current room).
Example 2: /list outcast (lists all users with the affiliation outcast to the current room, i.e. all banned users).
Example 3: /list moderator (list all users with moderator role, i.e. all owners, admins and temporary moderators present in the current room). Example 4: /list participant (lists all users that do not have a moderator role and are present in the room).

/ban {user}
Ban a user (insert the users nickname or JID).

/unban {jid}
Unban a user (insert the users JID). Please note: In members only rooms, you need to use /set affiliation member {jid} instead of /unban.

/kick {user}
Kick a user (insert the users nickname, not the JID).
