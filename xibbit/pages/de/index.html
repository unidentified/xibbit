XMPP (Jabber) ist ein Sofortnachrichtenprotokoll, das ähnliche Funktionen definiert wie z.B. ICQ, Skype oder Google Talk bieten, allerdings sehr flexibel. Es gibt mittlerweile viele XMPP Server, und eine große Anzahl von XMPP Clienten. Bei vielen Servern kann man sich kostenlos auf der Homepage der Betreiber registrieren. Viele unterstützen auch Gruppenchats.

Einige XMPP Server erlauben zudem Webzugriff. Das ermöglicht es, XMPP Clienten - vom Supportchat bis hin zum vollständigen Instant Messenger - auf einer Webseite laufen zu lassen.

Diese Webseite funktioniert als XMPP Client für Gruppenchats. Sie können Sie mit einem XMPP Accounts bei einem Server, der Gruppenchat und Webzugriff unterstützt, nutzen.
Wenn Sie einen XMPP Account haben, verwenden Sie bitte die Login-Seite, um zu dem Server zu verbinden. Beachten Sie, dass Sie einen XMPP Accounts zunächst auf der Homepage des Server registrieren müssen.

Verwenden Sie die FAQ um mehr Informationen zu XMPP und der Nutzung dieser Seite zu erhalten, und die Befehlsübersicht für eine Liste aller im Chat verwendbaren Befehle.

Auf dieser Seite können Sie zudem Links zum automatischen Betreten von Räumen erstellen. Geben Sie die Serverdetails an, und die Räume, die ein User, der auf den Link klickt, betreten soll.
User, die auf den Link klicken, erhalten eine Loginform, bei der sie Username (JID/ Jabber ID, in der Form username@server) und Passwort eingeben müssen. Sie benötigen einen Link beim verwendeten Server.
Manche Server unterstützen das unregistrierte Betreten (vorhandener) Räume. Dazu muss der Server für anonyme (unregistrierte) Logins angegeben werden. Erstellen Sie evtl. einen Link zum regstrierten und einen Link zum unregistrierten Betreten falls der Server dies unterstützt. User, die auf einen Link zum unregistrierten Betreten klicken, werden nur nach dem Usernamen gefragt und können einfach einen Nick der Wahl eingeben.
Sie können einen solchen Link bookmarken, oder verwenden, um sehr leicht und ohne zusätzliche Serverauslastung einen Chat zu Ihrer Webseite hinzuzufügen - geben Sie den Server an, bei dem die User sich registrieren können, und den Link zum automatischen Betreten der Räume als Chatlink.

Dieser XMPP Client unterstützt zudem clientseite Verschlüsselung. Das heißt, dass alle Nachrichten in einem Chatfenster mit dem im Link zum automatischen Betreten oder in der Loginform eingegebenen Passwort noch im Browser verschlüsselt werden - allerdings NUR die Nachrichten, nicht etwa Nicknames, Raumnamen, Einstellungen, Raumthema oder Sonstiges.
Das Verschlüsselungspasswort muss für alle Nutzer das selbe sein, und alle, die verschlüsselt Chatten wollen, müssen den Clienten auf dieser Seite verwenden (oder einen anderen Client, der die selbe Art der Verschlüsselung unterstützt - das ist allerdings aktuell nur diese Seite). Nicht oder anders verschlüsselte Nachrichten kommen gar nicht erst an.
Ein Verschlüsselungspasswort darf NIE gleichzeitig Loginpasswort oder Raumpasswort sein, da es sonst dem Server bekannt ist.

Falls Sie einen XMPP Multi User Chat Raum registrieren oder administrieren wollen, lesen Sie bitte die Infoseite. Dieser Client kann zur Raumverwaltung genutzt werden.
