Read here if you want to create your own chatroom and a link for users to autojoin it.
You can create a link for a room even if you do not moderate it as well, so make links for servers/ rooms you join often as well - you won't have to fill the forms each time you login, and can choose custom styles.

* [ ALL messages in this window, group and private, will be encrypted and decrypted using this password. Of course, all participants will have to use the same password. NEVER use the same password for encryption you already use as a room password, or as a login password, as these passwords are sent to the server - the encryption password never is and must not be. If you create a direct link for a room, please specify the encryption password if you use one. It will be included in the direct link. Using an encryption password is not mandatory, but can sometimes increase privacy, as only who knows the password or the link containing the password can read the messages (not even the server, as long as no server admin knows the password as well).]

** [ Some servers allow unregistered users to join. If choosing a server from list, these servers have a "(+a)" appended after the server name. Just leave the password field empty and enter a nickname of your choice. If entering server details yourself (not choosing from list), you need to specify the anonymous server to connect unregistered. If you're going to connect with a registered account, you can leave this field empty. ]
